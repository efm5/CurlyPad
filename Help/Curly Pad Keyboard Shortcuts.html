>Curly Pad Editor’s

Keyboard Shortcuts and menu Accelerator Keys

This is a list of the many default keyboard shortcuts and menu accelerator keys. Because >Curly Pad is designed to be Dragon-friendly, literally every menu action has a keyboard shortcut. Every menu item which may be opened to expose subitems has an accelerator key. Some subitems whose names may cause Dragon some confusion also have accelerator keys. Every item in every a dialog (Buttons, CheckBoxes, RadioButtons etc.) can be directly manipulated with Dragon. Every text and numeric entry field has a Label-like prefix Button — which, when exercised with Dragon, selects the field and selects everything in it, ready for dictation.
